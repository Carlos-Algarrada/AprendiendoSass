Encabezado h1 - Título Principal

Se usa para el título principal de la página. Debe ser único por página.

Encabezado h2 - Subtítulo Principal

Indica secciones importantes dentro de la página.

Encabezado h3 - Subsección

Se utiliza para subsecciones dentro de un `h2`.

Encabezado h4 - Subapartado

Menos relevante que `h3`, útil para subapartados más pequeños.

Encabezado h5 - Detalles adicionales

Se usa para detalles o información complementaria.

Encabezado h6 - Información secundaria

Es el nivel más bajo de importancia en los encabezados.
